# book-management-app

本管理アプリは[こちら](https://frontend-three-psi-89.vercel.app/)から

## 機能

- 買った本や読んだ本を管理
- 読書履歴から AI がおすすめの本を推薦
- 本の検索
- 無限スクロール
- 認証機能
  - 以下のアカウントでログインできます
    - メールアドレス：tmp@tmp
    - パスワード：tmp
